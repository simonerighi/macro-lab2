
# Introduzione alla Macroeconomia – DEMB

Web app didattica beta sviluppata da Simone Righi per il Dipartimento di Economia “Marco Biagi” dell’Università di Modena e Reggio Emilia.

## Avvio locale
```bash
npm install
npm run dev
```

## Deploy su GitHub Pages
Se il repository non si chiama `macro-lab`, modifica la riga `base` in `vite.config.ts`:

```ts
base: '/nome-repo/'
```

Poi esegui:

```bash
npm run deploy
```

## Nota
L'app implementa versioni lineari e didattiche dei modelli macroeconomici.
